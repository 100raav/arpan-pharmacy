# Arpan Pharmacy

A simple, offline‑first pharmacy management system built with HTML, CSS, and JavaScript.  
Data is stored locally in your browser using `localStorage`.

## Features
- Dashboard with real‑time charts (earnings, medicines sold, gender distribution)
- Medicine inventory management (add, delete, filter)
- Patient records with purpose and payment mode
- Billing with receipt printing
- Due payments tracking

## Live Demo
Check out the live version here:  
**[https://arpan-pharmacy.vercel.app/](https://arpan-pharmacy.vercel.app/)**  


## Technologies Used
- HTML5, CSS3, JavaScript (ES6)
- Bootstrap 5
- Font Awesome 6
- Chart.js
- LocalStorage for data persistence

## Deployment
This project is deployed on Vercel.
